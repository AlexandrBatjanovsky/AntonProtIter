Демо-версия COSMAP

Необходимые библиотеки перечислены libs.txt.
Для работы программы надо хотя бы 3 ГБ памяти видеокарты.

Содержит две нейросети:
    DHD - исходная архитектура Анны Хадорович. Оригинальных весов у меня не было, поэтому приходилось обучать самому.
        Обучена сеть здесь не лучшим образом, и реализовано только предсказывание интерфейса,
        но общее представление об даёт.

    FCN5 - лучшая из сетей, которые предсказывают матрицы косинусов. Алгоритм для неё реализован целиком - от чтения
        исходных данных, то полного предсказания структуры комплекса.

Прилагаются 42 .pkl файла содержащие 21 комплекс из обучающей выборки и 21 из тестовой.
Данные выборки не очень репрезентативны, на полных наборах данных результаты немного хуже [FCN5].


Для демонстрации работы DHD установите следующие параметры в конфигурационном файле new_cfg.json:
    "mode": "inspect"
    "net": "dhd"
    "attempt": {
        "checkpoint": "dhd0_at9/save.ckpt",
        ...
    }
    "inspect": {
        "data": "valid" или "train",
        "count": число от 0 до 21
    }
После этого запустите скрипт new_launcher.py


Для демонстрации работы FCN5 установите следующие параметры в конфигурационном файле new_cfg.json:
    "mode": "inspect"
    "net": "cosmap_fcn5"
    "attempt": {
        "checkpoint": "fcn5_at10/save.ckpt",
        ...
    }
    "inspect": {
        "data": "valid", "train" или "both",
        "count": число от 0 до 21
    }
После этого запустите скрипт new_launcher.py


Кроме того, для FCN5 можно полностью обработать обучающую/тестовую или обе выборки,
и собрать всю статистику в .csv файл. Для этого необходимо:
    "mode": "predict"
    "net": "cosmap_fcn5"
    "attempt": {
        "checkpoint": "fcn5_at10/save.ckpt",
        ...
    }
    "inspect": {
        "data": "valid", "train" или "both",
        ...
    }
После этого запустите скрипт new_launcher.py
Сохранённые результаты обработки 42 комплексов можно видеть в файле predict_fcn5_both.csv.
